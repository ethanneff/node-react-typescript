# Node React Typescript

## Why

A mono javascript repo for both client and server.

## What

- [x] typescript
- [x] react client
- [x] node/express server
- [x] jest testing
- [x] es linting
- [x] complete crud example
- [x] client build
- [ ] server build
- [ ] hosting

## How


#### Create

```sh
git clone git@github.com:ethanneff/node-react-typescript.git
cd node-react-typescript
yarn
```

#### Run

```sh
yarn client
```

```sh
yarn server
```

#### Test

```sh
yarn lint
```

```sh
yarn test
```

#### Deploy

```sh
yarn build
```
